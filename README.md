# アプリ名
SUMAU

## サイト概要
気になった物件情報を登録できるアプリ

### サイトテーマ
物件情報を登録することができるサイト


### テーマを選んだ理由
他社サイトに掲載されていない物件や、自分が気になった物件の情報を登録できるようにすることで、
ユーザーが物件探しなどの際に、比較検討できるようにしたいと思ったからです。

### ターゲットユーザ
物件を探している人

### 主な利用シーン
物件探し、気になった物件を把握したい時
